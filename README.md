# AquaLNCRpred

A web server for predicting long non-coding RNA (lncRNA) sequences in *Eriocheir sinensis* (Chinese Mitten Crab) and related aquaculture organisms.

```r
install.packages(c(

  "shinydashboard",
  "dashboardthemes",
  "markdown",

  "shinyjs",
  "protr",
  "randomForest",
  "caret",
  "tidyverse",
  "kernlab",
  "ftrCOOL",

))
