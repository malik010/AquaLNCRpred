# Downloads

## 1. Standalone Version of AquaLNCRpred

### Download and Installation

To use **AquaLNCRpred** as a standalone tool, click **[Download](AquaLNCRpred_standalone.tar.gz)** to obtain the package.

After the download is complete, uncompress the file `AquaLNCRpred_standalone.tar.gz`.

**On Linux:**

```bash
tar -xvzf AquaLNCRpred_standalone.tar.gz
```

**On Windows:**

To uncompress files with the `.tar.gz` extension on Windows, please refer to the appropriate extraction tool (e.g., 7-Zip or WinRAR).

Uncompressing the archive will create a directory named:

```text
AquaLNCRpred_standalone
```

Navigate to this directory and follow the instructions provided in the `README` file to install and run AquaLNCRpred.
